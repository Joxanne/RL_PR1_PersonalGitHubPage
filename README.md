# Joxanne's Personal GitHub Page

A modern, responsive, single-page personal portfolio featuring a glassmorphism design and dynamic elements.

## Features

- **Dynamic Background**: Animated, colorful floating blobs (Primary, Secondary, and Pink).
- **Glassmorphism UI**: A frosty, translucent container holding the main content with backdrop blurring and drop shadows.
- **Gradient Typography**: Elegant text rendering using Google Fonts (`Outfit` and `Inter`) with a shimmer gradient effect.
- **Live Clock & Date**: Real-time clock and full date display running on vanilla JavaScript.
- **Status Indicator**: An animated pulsing "Online" indicator pill.
- **Responsive**: Fully responsive and mobile-friendly layout.

## Live Demo

If GitHub Pages is enabled for this repository on the `main` branch, the site should be accessible at:
[https://joxanne.github.io/RL_PR1_PersonalGitHubPage/](https://joxanne.github.io/RL_PR1_PersonalGitHubPage/)

## Tech Stack

- **HTML5**: Semantic document structure
- **CSS3 (Vanilla)**: For the glassmorphism effects, flexbox layout, CSS variables, and all animations/keyframes.
- **JavaScript (Vanilla)**: Handles the time and date logic.

## Setup & Local Development

To run this project locally, simply clone the repository and open `index.html` in any modern web browser.

```bash
git clone https://github.com/Joxanne/RL_PR1_PersonalGitHubPage.git
```
Then double-click the `index.html` file. No build steps or package managers are required!
